# Task and Follow-Up Model

Last updated: 2026-07-18

Tasks represent operational work, not automation.

Default task types:

- submit_application
- review_resume
- follow_up
- prepare_recruiter_screen
- prepare_interview
- send_thank_you
- complete_assessment
- review_offer
- record_outcome
- other

Statuses: `open`, `completed`, `cancelled`, `overdue`.

The system tracks tasks but does not send emails or notifications in this sprint.
