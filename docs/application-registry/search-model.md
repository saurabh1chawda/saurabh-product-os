# Search Model

Last updated: 2026-07-18

Search and list commands support filters for:

- active status
- lifecycle stage
- application status
- company
- role
- archetype or role pack
- application date
- priority
- location
- Product OS module
- outcome
- tags
- overdue follow-up
- stale applications
- missing resume link
- missing JD link

Output modes are designed for readable terminal output and JSON. Sanitized output must not expose private contact details.
