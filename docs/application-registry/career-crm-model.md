# Career CRM Model

Last updated: 2026-07-18

Career CRM extends application tracking with contacts, tasks, notes, follow-ups, and interview operations. It is intentionally local-first and private.

## CRM Objects

- Application records.
- Contacts.
- Tasks.
- Notes.
- Events.

## CRM Questions

- What should be followed up today?
- Which applications are stalled?
- Which contacts are linked to active opportunities?
- Which interviews require preparation?
- Which records are missing Resume OS or JD traceability?
