# Career Console Component Library

## Components

### Career Shell

Provides sidebar navigation, top context, breadcrumbs, read-only status, and responsive layout.

### Operations Console

Renders action-first operational modules:

- Today's Actions
- Daily Brief
- Performance summary
- Pipeline overview
- Timeline
- Leaderboards
- Warnings
- Registry health
- Task center

### Application Table

Client-side search, filtering, sorting, pagination, and keyboard navigation over read-only data.

### Application Detail

Read-only application evidence view containing metadata, resume snapshot, JD snapshot, contacts, tasks, events, and outcome.

## Design Constraints

- Cards are used only for individual repeated items or compact panels.
- Components use existing Tailwind tokens.
- No decorative graphics, icons-only mystery controls, or non-operational visuals.
- No backend mutation controls are exposed.
