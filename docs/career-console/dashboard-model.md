# Career Console Dashboard Model

## Model Principle

This is not a reporting dashboard. Historical metrics are subordinate to operational next actions.

## Primary Surfaces

### Today's Actions

Includes follow-ups, overdue tasks, stale applications, missing recruiter links, missing resume traceability, and missing JD traceability.

### Pipeline Overview

Stages are grouped into:

- Applications
- Recruiter Screen
- Hiring Manager
- Technical
- Final
- Offers
- Rejected
- Withdrawn
- Accepted

### Performance Summary

Uses deterministic Application Intelligence metrics:

- Applications
- Response Rate
- Interview Rate
- Offer Rate
- Average Response Time
- Applications This Month
- Open Applications
- Closed Applications

### Leaderboards

Leaderboards rank resume variants, role archetypes, and companies by response, interview, offer, and timing metrics.

### Registry Health

Health indicators summarize registry validation, privacy validation, schema validation, traceability coverage, resume coverage, JD coverage, event consistency, and index status.
