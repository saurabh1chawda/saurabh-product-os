# Hi, I'm Saurabh Chawda.

This GitHub exists to share the working artifacts behind that practice.

Welcome to my Open Product Lab.

This is where I publish the frameworks, templates, decision systems, and reusable product artifacts behind [Product OS](https://saurabh-product-os.vercel.app).

## Executive Positioning

I build products and decision systems across AI, platforms, enterprise SaaS, payments, and growth.

Product OS explains how I think. GitHub shows the reusable assets behind that thinking.

Start with the [Executive Brief](https://saurabh-product-os.vercel.app/executive), then explore the repositories and product artifacts that support it.

## Operating Principle

> Every repository should leave another Product Manager better than they arrived.

That means each public repo should offer something useful: a sharper decision model, a reusable template, a practical framework, or a clearer way to connect customer value, technical choices, and business outcomes.

## Current Focus

- AI product decision systems for prioritization, architecture, trust, and measurement.
- Platform strategy artifacts for scalability, reliability, and AI readiness.
- Product Leadership Briefs that document decisions, trade-offs, and outcomes.
- Toolkits and templates that other product teams can reuse.
