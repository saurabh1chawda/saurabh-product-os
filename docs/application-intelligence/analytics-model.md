# Application Intelligence Analytics Model

Last updated: 2026-07-17

## North Star

Qualified Interview Rate.

## Portfolio Metrics

- Applications
- Response rate
- Interview rate
- Offer rate
- Average response time
- Applications per archetype
- Applications per company
- Applications per role
- Applications per resume version
- Applications per role pack
- Applications per Product OS module

## Dashboard Specification

Sections:

- Pipeline
- Conversion Funnel
- Response Times
- Applications
- Interview Rate
- Offer Rate
- Role Distribution
- Company Distribution
- Override Analytics
- Resume Performance
- Trend Analysis

## Insight Engine

Insights are deterministic and must include evidence. Examples:

- Most successful headline.
- Most successful summary.
- Highest-performing archetype.
- Highest-performing project.
- Most overridden bullet.
- Fastest response company.
- Longest interview funnel.
- Highest interview conversion.

No insight should be generated from unavailable or insufficient data.

## Experiment Framework

Supported experiment examples:

- Headline A vs Headline B.
- Summary A vs Summary B.
- Project included vs removed.
- Product OS module variants.

Each experiment must track variant, sample size, funnel outcome, and caveats.
