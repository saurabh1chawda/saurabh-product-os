# Skills and Tools Assembly

## Ordering

Skills are ordered by:

1. Must-cover JD competencies.
2. Primary archetype.
3. Secondary archetypes.
4. Evidence support.
5. ATS importance.
6. Interview defensibility.

## Rules

- Every skill must be supported.
- Do not add a tool merely because the JD mentions it.
- Separate product, domain, technical, and leadership skills.
- Avoid broad skill inventories.
- Prefer 12-20 high-relevance skills.
- Tools are secondary to competencies.
- Remove obsolete tools unless role-relevant.
- Do not present familiarity as deep expertise.
