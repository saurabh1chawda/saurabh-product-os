# Credential Selection

## Include When Relevant

- Certifications.
- Awards.
- Publications.
- Education.

## Rules

- Education is immutable.
- Awards support leadership and execution credibility.
- Publications are included when domain-relevant.
- Certifications are included when role-relevant.
- Low-value certifications must not displace production evidence.
- Expired or unverifiable credentials are excluded.
- Optional credential sections stay compact.

## Default Handling

- Education is always included.
- Awards are included when space allows.
- Certifications are included when role-relevant.
- Publications are included only when they strengthen the target archetype or domain.
