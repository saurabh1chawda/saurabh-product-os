# ATS Guidelines

Last updated: 2026-07-17

## Required

- Single-column layout.
- Plain text headings.
- Editable DOCX.
- Searchable PDF.
- No tables.
- No graphics.
- No decorative icons.
- Standard bullet paragraphs.
- Readable URLs.
- Copy/paste-safe text.

## Avoid

- Text boxes.
- Header-only contact information.
- Image-based PDFs.
- Hidden text.
- Unusual fonts.
- Multi-column layouts.
- Decorative separators that parse as content.

## Export Rule

ATS compatibility is allowed to constrain formatting, but never content. If content does not fit, the exporter warns and waits for human review instead of rewriting.
