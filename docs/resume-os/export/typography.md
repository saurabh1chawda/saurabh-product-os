# Typography

Last updated: 2026-07-17

## Font System

- Font family: Arial.
- Body size: 10.5 pt.
- Name/header size: 16 pt.
- Headline size: 11 pt.
- Section headings: 11 pt, bold.
- Bullet text: 10.5 pt.

## Paragraph Rhythm

- Body line spacing: 1.0.
- Paragraph spacing after body blocks: 3 pt.
- Section spacing before: 7 pt.
- Bullet spacing after: 1 pt.

## Emphasis

- Bold only for name and section headings.
- Italics are not used by default.
- Links are underlined and blue in DOCX; PDF preserves link text and annotations.

## Alignment

- Header: centered.
- Body: left aligned.
- Bullets: left aligned with standard indent.

## Page Control

Widow/orphan handling is best effort in DOCX. The QA engine detects orphan headings and likely overflow; the renderer does not rewrite content to resolve them.
