# Formatting Rules

Last updated: 2026-07-17

## Layout

- Single column.
- No tables.
- No text boxes.
- No floating graphics.
- No icons.
- US Letter page size.
- Conservative margins.
- Standard ATS-readable section headings.

## Section Order

1. Header
2. Target headline
3. Professional summary
4. Experience
5. Selected project or Product OS proof
6. Skills
7. Certifications
8. Education
9. Awards, if present

## Spacing

- Tight but readable section spacing.
- Bullets use consistent indentation.
- No decorative dividers.
- No columns or sidebars.

## Hyperlinks

Links are preserved for LinkedIn, GitHub, Product OS, portfolio URLs, and email where present. Link labels must remain readable when copied as plain text.
