# Professional Summary Modules

## Purpose

Summary modules provide reusable, role-specific opening summaries. They are not final resume summaries; they are controlled building blocks.

## Rules

- 70 to 90 words maximum when assembled.
- Include only verified metrics.
- Use one Product OS sentence maximum.
- Avoid repeating the job description.
