# JD Intelligence Validation Matrix

## Acceptance Criteria

- Primary archetype correct for all clear fixtures.
- Hybrid fixture identifies appropriate secondary archetypes.
- No unsupported achievement recommended.
- No simulation treated as production.
- No orphan bullet recommended.
- No fabricated keyword introduced.
- Evidence ranking is explainable.
- Gaps are reported honestly.
