# Bullet Narrative Rules

## Goals

- Make impact visible.
- Preserve exact metrics.
- Improve verb diversity.
- Keep technical evidence understandable.
- Keep bullets parallel and skimmable.

## Verb Guidance

Avoid overusing:

- Led
- Drove
- Owned
- Leveraged
- Spearheaded

Preferred measured verbs:

- Generated
- Built
- Scaled
- Improved
- Partnered
- Prioritized
- Converted
- Established
- Increased
