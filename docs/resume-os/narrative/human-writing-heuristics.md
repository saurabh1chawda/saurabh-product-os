# Human Writing Heuristics

## Scored Dimensions

| Heuristic | Scoring method |
| --- | --- |
| Sentence variety | Penalize repeated openings and identical structures. |
| Sentence length | Reward 12-28 word sentences; penalize overly long chains. |
| Verb diversity | Penalize repeated generic verbs. |
| Metric distribution | Reward visible but not crowded metrics. |
| Transition quality | Reward logical movement from role fit to evidence. |
| Redundancy | Penalize repeated domain phrases. |
| Parallelism | Reward consistent bullet structure within sections. |
| Cadence | Reward varied rhythm without losing scanability. |
| Executive tone | Reward measured, evidence-led phrasing. |
| Specificity | Reward concrete nouns, metrics, and product domains. |

## Score Bands

- 95-100: publication-ready after factual review.
- 90-94: strong human-quality draft.
- 85-89: structurally sound but needs polish.
- Below 85: narrative revision required.
