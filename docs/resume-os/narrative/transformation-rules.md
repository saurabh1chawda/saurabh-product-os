# Narrative Transformation Rules

| Rule ID | Name | Allowed action | Risk |
| --- | --- | --- | --- |
| NAR-HEAD-001 | Headline compression | Remove redundant domain terms while preserving role fit | Low |
| NAR-SUM-001 | Summary lead improvement | Start with role-relevant value proposition | Low |
| NAR-SUM-002 | Metric clustering | Group metrics into a readable business-impact sentence | Medium |
| NAR-BLT-001 | Impact-first bullet | Move measurable outcome earlier without changing meaning | Medium |
| NAR-BLT-002 | Verb diversity | Replace repeated verbs with equivalent precise verbs | Low |
| NAR-BLT-003 | Clause tightening | Remove filler or repeated clauses | Low |
| NAR-BLT-004 | Technical clarity | Clarify technical evidence without adding technology | Medium |
| NAR-TONE-001 | Executive tone | Replace generic leadership phrasing with measured product language | Low |

## Reversal

Every rule stores original and transformed text. Reversal means replacing transformed text with the original text from the transform report.
