## Metric Rules

- Do not create derivative metrics without approval.
- Do not translate Rs. values into another currency without a source.
- Do not use simulation metrics from LoyaltyIQ as real business outcomes.
- If a metric appears with different wording across sources, prefer Product OS wording for public evidence and reference-resume wording for application-specific phrasing.
