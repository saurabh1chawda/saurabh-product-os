## Usage Rules

- Every resume bullet must map to exactly one achievement ID.
- If a bullet combines multiple achievements, split it into separate bullets or create a new approved composite record.
- Metrics must appear exactly as approved in `data/achievements.yaml`.
- Simulation-only project evidence must not be listed as employment achievement.
