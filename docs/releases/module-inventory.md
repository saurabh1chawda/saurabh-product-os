## Purpose

This inventory documents the major Career OS modules included in the v1.0.0 pilot baseline. It is a release-management reference for understanding module ownership, responsibilities, dependencies, public interfaces, and validation status before real-world pilot use.

## Release Notes

- Module versions are aligned to `v1.0.0-pilot` for the first real-world pilot baseline.
- Pilot readiness means the module is usable for controlled real-world validation by one candidate.
- Pilot readiness does not imply multi-user production support, external integrations, or public SaaS availability.
- Future roadmap decisions should be based on observed pilot gaps, defects, and workflow friction.
