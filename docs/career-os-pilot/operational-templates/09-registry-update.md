# Registry Update Template

## Purpose

Ensure the Application Registry reflects the submitted application and artifact links.

## Required Fields

- Registry update ID: `[REG-UPD-YYYY-###]`
- Application ID: `[APP-YYYY-###]`
- Update date: `[YYYY-MM-DD]`
- Current stage: `[Submitted / recruiter screen / interview / terminal]`
- Current status: `[Open / closed / waiting / action required]`
- Priority: `[High / medium / low]`
- Resume variant ID: `[RESUME-YYYY-###]`
- JD Snapshot ID: `[JD-YYYY-###]`
- Source/channel: `[Channel]`
- Next action or valid waiting state: `[Action or waiting state]`
- Traceability complete: `[Yes / No]`

## Optional Fields

- Tags: `[Tag list]`
- Contact reference: `[Private reference if needed]`
- Notes reference: `[Private reference if needed]`

## Required Evidence

- Application submission record.
- Resume and JD artifact references.
- Registry validation result where available.

## Completion Checklist

- [ ] Registry record exists.
- [ ] Resume link is present.
- [ ] JD link is present.
- [ ] Stage and status are consistent.
- [ ] No private data is committed.

## Example Placeholders

- Application ID: `APP-2026-001`
- Current status: `[Open]`
