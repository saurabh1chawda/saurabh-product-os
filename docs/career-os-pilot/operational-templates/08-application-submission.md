# Application Submission Template

## Purpose

Record manual submission readiness and completion without automating external actions.

## Required Fields

- Submission ID: `[SUB-YYYY-###]`
- Application ID: `[APP-YYYY-###]`
- Resume variant ID: `[RESUME-YYYY-###]`
- JD Snapshot ID: `[JD-YYYY-###]`
- Submission date: `[YYYY-MM-DD]`
- Submission channel: `[Company site / referral / recruiter / job board]`
- Application URL or reference: `[URL or local private reference]`
- Confirmation received: `[Yes / No / Not available]`
- Gate 4 decision: `[Submit / defer / cancel]`
- Initial follow-up plan: `[Date or valid waiting state]`

## Optional Fields

- Referral note: `[Private if needed]`
- Portal-specific requirements: `[Notes]`

## Required Evidence

- Approved QA Checklist.
- Resume export or approved draft.
- Submission source.
- Confirmation where available.

## Completion Checklist

- [ ] External submission was manual.
- [ ] Submission date is recorded.
- [ ] Application Registry update is required next.
- [ ] Follow-up or valid waiting state is defined.

## Example Placeholders

- Submission channel: `[Company site]`
- Confirmation received: `[Yes]`
