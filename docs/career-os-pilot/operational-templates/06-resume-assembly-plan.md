# Resume Assembly Plan Template

## Purpose

Plan the structure of the resume draft without changing canonical facts or approved strategy.

## Required Fields

- Resume Plan ID: `[PLAN-YYYY-###]`
- Resume Strategy ID: `[STRAT-YYYY-###]`
- Resume variant ID: `[RESUME-YYYY-###]`
- Target file name: `[Saurabh-Chawda_Company_Role_YYYY-MM-DD_vX]`
- Section order: `[Ordered section list]`
- Headline module: `[Component ID or text reference]`
- Summary module: `[Component ID]`
- Experience role blocks: `[Role block list]`
- Selected bullet IDs: `[Bullet IDs]`
- Selected project module: `[Project ID or not applicable]`
- Skills modules: `[Module IDs]`
- Product OS links: `[Approved links]`
- Estimated page count: `[One / two / overflow risk]`

## Optional Fields

- Alternate bullets excluded: `[Bullet IDs and reason]`
- Manual review focus: `[Notes]`

## Required Evidence

- Approved Resume Strategy.
- Bullet-to-achievement mappings.
- Link list.

## Completion Checklist

- [ ] Every selected bullet has one achievement ID.
- [ ] Section order is ATS-safe.
- [ ] Two-page estimate is acceptable.
- [ ] No unsupported claims are introduced.

## Example Placeholders

- Resume variant ID: `RESUME-2026-001`
- Estimated page count: `[Two]`
