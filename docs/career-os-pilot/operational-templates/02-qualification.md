# Qualification Template

## Purpose

Determine whether an opportunity should proceed to JD Snapshot and JD Intelligence. This template supports Gate 1 in [Decision Gates](../decision-gates.md).

## Required Fields

- Opportunity ID: `[OPP-YYYY-###]`
- Qualification date: `[YYYY-MM-DD]`
- Company: `[Company name]`
- Role title: `[Role title]`
- Target role fit: `[Strong / Medium / Weak]`
- Location/work model fit: `[Yes / No / Needs review]`
- Experience relevance: `[Strong / Medium / Weak]`
- Application willingness: `[Yes / No]`
- Duplicate application: `[Yes / No]`
- Known closed role: `[Yes / No]`
- Gate 1 decision: `[Proceed / reject / hold]`
- Decision owner: `[Saurabh Chawda]`

## Optional Fields

- Referral possibility: `[Yes / No / Unknown]`
- Company priority: `[High / Medium / Low]`
- Risk notes: `[Privacy-safe notes]`

## Required Evidence

- Opportunity intake.
- Role posting or equivalent source.
- Qualification reasoning.

## Completion Checklist

- [ ] Minimum eligibility is checked.
- [ ] Proceed/reject decision is recorded.
- [ ] Rejection reason is privacy-safe if rejected.
- [ ] Next stage is clear.
