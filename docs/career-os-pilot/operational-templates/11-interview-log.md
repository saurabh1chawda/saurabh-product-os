# Interview Log Template

## Purpose

Track interview scheduling, completion, and outcomes while protecting confidential interview content.

## Required Fields

- Interview ID: `[IV-YYYY-###]`
- Application ID: `[APP-YYYY-###]`
- Interview stage: `[Recruiter / hiring manager / technical / final]`
- Scheduled date: `[YYYY-MM-DD]`
- Format: `[Phone / video / onsite / take-home]`
- Preparation task: `[Task reference]`
- Completion status: `[Scheduled / completed / canceled / rescheduled]`
- Outcome status: `[Pending / advanced / rejected / no decision / withdrawn]`
- Gate 5 decision: `[Continue / follow up / wait / record outcome]`
- Next action: `[Action or valid waiting state]`

## Optional Fields

- Interviewer role: `[Role only; no unnecessary personal data]`
- Privacy-safe preparation notes: `[Notes]`
- Follow-up date: `[YYYY-MM-DD]`

## Required Evidence

- Interview invitation or scheduling confirmation.
- Completion event.
- Explicit outcome if available.

## Completion Checklist

- [ ] Confidential interview questions are not stored.
- [ ] Completion or reschedule status is recorded.
- [ ] Outcome is explicit or marked pending.
- [ ] Next action is clear.

## Example Placeholders

- Interview stage: `[Hiring Manager]`
- Outcome status: `[Pending]`
