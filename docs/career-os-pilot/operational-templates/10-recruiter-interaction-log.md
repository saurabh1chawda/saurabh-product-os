# Recruiter Interaction Log Template

## Purpose

Record privacy-safe recruiter interactions and next actions without storing unnecessary raw communication.

## Required Fields

- Interaction ID: `[INT-YYYY-###]`
- Application ID: `[APP-YYYY-###]`
- Interaction date: `[YYYY-MM-DD]`
- Interaction type: `[Inbound / outbound / follow-up / screen scheduling]`
- Contact reference: `[Private reference or anonymized label]`
- Stage before interaction: `[Stage]`
- Stage after interaction: `[Stage]`
- Explicit employer response: `[Yes / No]`
- Next action: `[Action or valid waiting state]`
- Privacy handling: `[Raw message stored? Yes / No; reason if yes]`

## Optional Fields

- Follow-up due date: `[YYYY-MM-DD]`
- Summary: `[Privacy-safe summary]`

## Required Evidence

- Employer response or interaction record.
- Stage transition reason if stage changed.
- Next action.

## Completion Checklist

- [ ] No unnecessary raw email content is stored.
- [ ] No inferred rejection is recorded from silence.
- [ ] Stage transition is explicit.
- [ ] Next action is clear.

## Example Placeholders

- Interaction type: `[Inbound]`
- Stage after interaction: `[Recruiter Screen]`
