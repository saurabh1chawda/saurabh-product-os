# Opportunity Intake Template

## Purpose

Capture a potential role before qualification. This template supports the Opportunity stage in the [Career OS Pilot Workflow](../pilot-workflow.md).

## Required Fields

- Opportunity ID: `[OPP-YYYY-###]`
- Capture date: `[YYYY-MM-DD]`
- Company: `[Company name]`
- Role title: `[Role title]`
- Source/channel: `[Source]`
- Source URL or local reference: `[URL or path]`
- Location/work model: `[Location / remote / hybrid / onsite]`
- Initial role archetype guess: `[Archetype]`
- Duplicate check completed: `[Yes / No]`
- Initial owner decision: `[Review / reject]`

## Optional Fields

- Referral source: `[Name or channel, if appropriate]`
- Recruiter or contact known: `[Yes / No]`
- Posting ID: `[Posting ID]`
- Notes: `[Privacy-safe notes]`

## Required Evidence

- Role posting or credible source reference.
- Date the opportunity was discovered.
- Confirmation that no duplicate active application exists.

## Completion Checklist

- [ ] Opportunity is real.
- [ ] Role source is captured.
- [ ] Duplicate check is complete.
- [ ] No private recruiter or compensation details are included unnecessarily.
- [ ] Next stage is Qualification or rejection.

## Example Placeholders

- Opportunity ID: `OPP-2026-001`
- Company: `[Company]`
- Role title: `[Senior Product Manager]`
- Source URL: `[https://example.com/job-posting]`
