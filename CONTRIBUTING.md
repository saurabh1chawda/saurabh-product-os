# Contributing

Status: limited contribution model  
Last updated: July 2026

Product Leadership Platform is currently maintained as Saurabh Chawda's canonical Product OS and public product leadership evidence system.

External contributions are intentionally limited.

## What Is Welcome

- Documentation corrections.
- Broken link reports.
- Accessibility issues.
- Typo fixes.
- Suggestions that improve clarity of reusable frameworks or templates.

## What Is Not Accepted

- Edits to Saurabh's personal career positioning.
- Changes to company-specific metrics or Product Leadership Brief evidence.
- Rewrites of Product OS positioning.
- New features without prior discussion.
- Large architecture changes.

## Before Opening An Issue

Please check:

- The relevant Product OS page.
- The related README or documentation file.
- Existing documentation in `docs/`.

## Contribution Principle

Contributions should make the platform clearer, more accurate, or easier to use without changing the source-of-truth narrative.
