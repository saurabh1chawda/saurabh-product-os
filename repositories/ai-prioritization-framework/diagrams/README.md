# Diagram Inventory

Status: v1 visual source layer  
Last updated: July 2026

This directory holds Mermaid source files for diagrams that make AI prioritization easier to explain in roadmap and executive discussions.

Available diagrams:

- [AI Prioritization Flow](ai-prioritization-flow.mmd)
- [AI Decision Matrix](ai-decision-matrix.mmd)
- [Scoring Worksheet](scoring-worksheet.mmd)

## Rendering Guidance

These files use Mermaid syntax and can render in:

- GitHub Markdown previews.
- Mermaid Live Editor.
- Documentation sites with Mermaid enabled.
- Internal wiki tools that support Mermaid.

Recommended export formats:

- SVG for documentation.
- PNG for presentations.
- Mermaid source for repository maintenance.

Diagram standards:

- Explain one decision at a time.
- Avoid decorative AI imagery.
- Use executive-friendly labels.
- Keep diagrams readable on mobile.
- Link every diagram to the framework or template it supports.
