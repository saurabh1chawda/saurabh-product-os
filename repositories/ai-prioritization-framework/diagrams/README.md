# Diagram Inventory

Status: Planned visual layer

This directory will hold diagrams that make AI prioritization easier to explain in roadmap and executive discussions.

Planned diagrams:

- AI Prioritization Matrix

- Opportunity-to-Recommendation Decision Tree

Diagram standards:

- Explain one decision at a time.
- Avoid decorative AI imagery.
- Use executive-friendly labels.
- Keep diagrams readable on mobile.
- Link every diagram to the framework or template it supports.
