# Architecture

Status: v1 foundation  
Last updated: July 2026

## System Overview

AI Prioritization Framework is a documentation-first product repository. It is organized around a reusable framework, templates, examples, diagrams, and governance docs that help teams evaluate AI opportunities during roadmap planning.

## Folder Responsibilities

| Folder | Responsibility |
| --- | --- |
| `framework/` | Core framework, scoring model, criteria, decision tree, and anti-patterns |
| `templates/` | Reusable canvases, scoring sheets, and workshop materials |
| `examples/` | Concise examples showing framework application |
| `diagrams/` | Planned visual models for executive communication |
| `assets/` | Public screenshots, exports, and supporting assets |
| `docs/` | Architecture, governance, roadmap, and product specification |

## Content Model

Each framework document should help teams make a prioritization decision.

Each template should be reusable in roadmap planning.

Each example should show framework application without duplicating Product Leadership Briefs.

## Product OS Relationship

Product OS explains the executive Decision System. This repository provides the operational framework.

Do not copy long Product OS narratives into this repository. Link to Product OS for context and keep this repository focused on practical artifacts.
