# Downloads

Status: v1 download plan  
Last updated: July 2026

## Purpose

This document defines supported and future download formats for the AI Prioritization Framework.

The repository is currently Markdown-first. Downloadable assets should make the framework easier to use in roadmap workshops, AI investment reviews, product discovery, and executive presentations.

## Supported Now

| Asset | Format | Source |
| --- | --- | --- |
| AI Opportunity Canvas | Markdown | `templates/ai-opportunity-canvas.md` |
| Scoring Sheet | Markdown | `templates/scoring-sheet.md` |
| Workshop Template | Markdown | `templates/workshop-template.md` |
| AI Prioritization Flow | Mermaid | `diagrams/ai-prioritization-flow.mmd` |
| AI Decision Matrix | Mermaid | `diagrams/ai-decision-matrix.mmd` |
| Scoring Worksheet Flow | Mermaid | `diagrams/scoring-worksheet.mmd` |
| Logix Opportunity Card | Markdown | `examples/opportunity-cards/logix-ai-ready-platform.md` |

## Recommended Export Targets

### Markdown

Use as the canonical source format.

### CSV

Use for scoring tables and opportunity comparison.

Recommended export tables:

- Criterion Scoring
- Weighted Scoring
- Decision Conditions

### Excel

Use for roadmap workshops that compare multiple opportunities.

Recommended tabs:

- Opportunity Summary
- Criterion Scoring
- Weighted Scoring
- Recommendation
- Decision Log

### Google Sheets

Use for collaborative scoring sessions.

Recommended tabs match Excel.

### Notion

Future target.

Recommended Notion structure:

- Opportunity database
- Criteria scoring database
- Recommendation view
- Decision log relation

## Future Assets

- Spreadsheet scoring workbook.
- One-page executive summary template.
- PNG and SVG diagram exports.
- Workshop facilitation deck.
- Notion-ready scoring database.
