# Roadmap

Status: v1 foundation  
Last updated: July 2026

## Version 1: Framework Foundation

Goal: Establish the AI Prioritization Framework as a usable decision system for roadmap planning.

Scope:

- Flagship README
- Core framework
- Scoring model
- Evaluation criteria
- Decision tree
- Anti-patterns
- AI opportunity canvas
- Scoring sheet
- Workshop template
- Example structure
- Architecture, governance, and product specification docs

Exit criteria:

- A Product Manager can understand and apply the framework without reading Product OS.
- Teams can score an AI opportunity and make a recommendation.
- Anti-patterns are clear enough to stop weak AI ideas.

## Version 2: Templates and Facilitation

Goal: Turn the repository into a workshop-ready prioritization toolkit.

Scope:

- Downloadable formats
- Workshop facilitation notes
- More worked examples
- Portfolio comparison guidance
- Executive summary template

Exit criteria:

- A product team can run a prioritization workshop using the repository.
- Multiple AI opportunities can be compared consistently.
- Examples show how recommendations differ by context.

## Version 3: Interactive Tools

Goal: Make AI prioritization easier to use in planning and executive review.

Scope:

- Lightweight scoring worksheet
- Opportunity comparison tool
- Executive prioritization summary generator
- Optional exports or planning-tool integrations

Exit criteria:

- Teams can evaluate, compare, and summarize AI opportunities quickly.
- Interactive tools remain transparent and easy to inspect.
