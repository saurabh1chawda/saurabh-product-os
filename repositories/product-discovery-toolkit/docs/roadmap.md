# Roadmap

Status: v1 foundation  
Last updated: July 2026

## Version 1: Toolkit Foundation

Goal: Establish Product Discovery Toolkit as a usable discovery starter kit.

Scope:

- Flagship README
- Discovery Process
- Opportunity Assessment
- Evidence Ladder
- Prioritization Readiness
- Discovery Anti-Patterns
- Seven reusable templates
- Example structure
- Architecture, governance, and product specification docs

Exit criteria:

- A Product Manager can run early discovery using the toolkit.
- Evidence quality is clearly explained.
- Templates can be copied into real discovery work.
- Examples show discovery influence without duplicating Product OS.

## Version 2: Workshop Usability

Goal: Make the toolkit easier to facilitate with teams.

Scope:

- Facilitation notes
- Industry-specific discovery variants
- Evidence scoring improvements
- Customer synthesis examples
- Workshop agenda templates

Exit criteria:

- Teams can run a discovery workshop using the repository.
- Evidence can be compared consistently across opportunities.
- Discovery outputs can feed prioritization discussions.

## Version 3: Complete Discovery Operating Kit

Goal: Expand the toolkit into an end-to-end discovery operating system.

Scope:

- Editable formats
- End-to-end discovery workflow
- AI, SaaS, platform, fintech, and growth examples
- Lightweight discovery readiness tool

Exit criteria:

- A team can move from signal to validated opportunity using the toolkit.
- Artifacts remain simple, practical, and easy to adapt.
