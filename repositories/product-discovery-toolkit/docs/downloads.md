# Downloads

Status: v1 download plan  
Last updated: July 2026

## Purpose

This document defines supported and future download formats for the Product Discovery Toolkit.

The repository is currently Markdown-first. Downloadable assets should make the toolkit easier to use in customer discovery, user research, roadmap planning, and product workshops.

## Supported Now

| Asset | Format | Source |
| --- | --- | --- |
| Customer Interview Guide | Markdown | `templates/customer-interview-guide.md` |
| Discovery Canvas | Markdown | `templates/discovery-canvas.md` |
| Opportunity Solution Tree | Markdown | `templates/opportunity-solution-tree.md` |

## Export Targets

### Markdown

Use as the canonical source format.

### CSV

Use for interview synthesis, assumption mapping, validation scoring, and opportunity comparison.

### Excel

Use for discovery synthesis across multiple customers or segments.

Recommended tabs:

1. Interview Notes
2. Evidence Inventory
3. Assumptions
4. Validation Scorecard
5. Decision Log

### Google Sheets

Use for collaborative discovery synthesis.

Recommended tabs match Excel.

### Notion

Future target for team workspaces.

Recommended structure:

- Customer interview database.
- Opportunity database.
- Assumption database.
- Experiment database.
- Weekly discovery review view.

### Miro

Future target for workshops.

Recommended boards:

- Opportunity Solution Tree.
- Assumption Map.
- Evidence Ladder.
- Discovery Decision Tree.

## Future Assets
