# Downloads

Status: v1 download guidance  
Last updated: July 2026

## Purpose

Decision memos must be easy to move into the tools teams already use for executive reviews, product councils, roadmap planning, and investment committees.

The repository is currently Markdown-first. Downloadable formats should preserve the memo structure without adding presentation overhead.

## Supported Export Targets

### Markdown

Primary source of truth. Best for GitHub, internal repos, version control, and review comments.

### Google Docs

Recommended for collaborative executive review. Preserve headings, tables, and decision request sections.

### Microsoft Word

Recommended for formal leadership documents, customer-facing decision records, or offline review.

### PDF

Recommended for read-only executive packets after the decision memo is finalized.

### Notion

Recommended for lightweight internal product operating systems. Keep the decision request and final decision near the top.

### Confluence

Future target for enterprise teams. Recommended structure should preserve headings, tables, owners, and review dates.

## Export Standards

- Keep the recommendation visible above the fold.
- Preserve tables for evidence, alternatives, risks, and success metrics.
- Include status, date, owner, and confidence.
- Avoid turning memos into slide decks unless a meeting requires it.
- Link back to the Markdown source when possible.
