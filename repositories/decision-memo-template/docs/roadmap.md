# Roadmap

Status: v1 foundation  
Last updated: July 2026

## Version 1: Memo System Foundation

Goal: Establish Decision Memo Template as a reusable writing system for executive product decisions.

Scope:

- Flagship README
- Decision writing frameworks
- Decision Writing Pyramid
- Evidence standards
- Common memo mistakes
- Six reusable templates
- Example structure
- Architecture, governance, and product specification docs

Exit criteria:

- A Product Manager can write a concise decision memo using the repository.
- Templates are copyable.
- Frameworks explain executive communication quality.
- Examples show communication focus without duplicating Product OS.

## Version 2: Decision Review Kit

Goal: Help teams review and improve decision memos.

Scope:

- Completed sample memos
- Decision review checklist
- Decision log template
- Domain-specific memo variants
- Facilitation guide for decision reviews

Exit criteria:

- A team can review product decisions consistently.
- Decision logs can preserve context over time.
- Examples show what strong memo writing looks like.

## Version 3: Portable Formats

Goal: Make the templates usable across common team workflows.

Scope:

- Markdown versions
- Notion-ready versions
- Document-ready versions
- Decision quality scorecard
- Executive summary generator outline
- Team operating guide for written decisions

Exit criteria:

- Teams can adopt the memo format in their existing tools.
- Decision quality can be evaluated consistently.
