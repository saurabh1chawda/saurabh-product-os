# Architecture

Status: v1 foundation  
Last updated: July 2026

## System Overview

Decision Memo Template is a documentation-first product repository. It is organized around decision-writing frameworks, reusable memo templates, communication-focused examples, diagrams, and governance docs.

## Folder Responsibilities

| Folder | Responsibility |
| --- | --- |
| `frameworks/` | Decision writing, executive summary, stakeholder alignment, evidence, and mistake-avoidance frameworks |
| `templates/` | Copyable memo templates for executive, product, AI, platform, roadmap, and one-page decisions |
| `examples/` | Communication examples and compact decision cards |
| `diagrams/` | Mermaid visual models for teaching decision writing |
| `assets/` | Public previews, exports, and supporting assets |
| `docs/` | Architecture, governance, roadmap, product specification, downloads, and playbooks |

## Content Model

Each framework should improve decision communication.

Each template should be short enough for real teams to adopt.

Each example should focus on how the decision would be communicated, not on re-telling the full historical narrative.

## Product OS Relationship

Product OS explains disciplined product thinking. This repository provides the written artifact teams can use to practice that discipline.

Do not copy long Product OS narratives into this repository. Link to Product OS for context and keep this repository focused on executive communication.
