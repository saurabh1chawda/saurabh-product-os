This directory holds Mermaid source files that make decision writing easier to teach, review, and reuse.

Available diagrams:

- [Decision Writing Pyramid](decision-writing-pyramid.mmd)
- [Decision Flow](decision-flow.mmd)
- [Stakeholder Alignment Map](stakeholder-alignment-map.mmd)

## Rendering Guidance

These files use Mermaid syntax and can render in:

- GitHub Markdown previews.
- Mermaid Live Editor.
- Documentation sites with Mermaid enabled.
- Internal wiki tools that support Mermaid.

Recommended export formats:

- SVG for documentation.
- PNG for presentations.
- Mermaid source for repository maintenance.

Diagram standards:

- Clarify executive communication.
- Avoid decorative visuals.
- Keep labels short.
- Link each diagram to a framework or template.
