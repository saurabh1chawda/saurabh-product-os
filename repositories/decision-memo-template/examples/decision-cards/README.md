# Decision Cards

Status: v1 card format  
Last updated: July 2026

Decision cards summarize one product decision in a compact, reusable format.

Use a decision card when the full memo exists elsewhere but the team needs a fast way to review the decision logic.

## Card Format

- Decision:
- Context:
- Recommendation:
- Alternatives:
- Evidence:
- Trade-off:
- Confidence:
- Outcome:
- Related memo / brief:
