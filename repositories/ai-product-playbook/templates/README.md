# Template Index

Status: Planned template layer  
Last updated: July 2026

This directory will contain reusable AI product templates. The v1 goal is to define the template system before publishing full downloadable artifacts.

| Template | Purpose | Status |
| --- | --- | --- |
| AI PRD | Define an AI product opportunity, workflow, assumptions, risks, evaluation plan, and launch criteria | Planned |
| AI Experiment Canvas | Design experiments that answer one product decision before scaling investment | Planned |
| AI Opportunity Assessment | Evaluate whether an opportunity is a strong AI fit or a simpler product problem | Planned |
| AI Success Metrics Canvas | Connect model, system, workflow, customer, and business metrics | Planned |
| Prompt Review Checklist | Review prompts, context, failure modes, user control, and quality expectations | Planned |

## Template Standard

Each template should include:

- Purpose
- When to use it
- Inputs required
- Completion steps
- Review checklist
- Example output
- Related framework
- Product OS reference link
