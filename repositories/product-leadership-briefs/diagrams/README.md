# Diagram Inventory

Status: Planned visual layer  
Last updated: July 2026

This directory will hold reusable diagrams that make Product Leadership Briefs easier to understand and apply.

Planned diagrams:

- Product Decision Framework
- Situation, Complication, Question flow
- Strategic Options Matrix
- Trade-off Map
- Stakeholder Alignment Map
- Impact Dashboard pattern
- Product Principle extraction flow

Diagram standards:

- Make the decision easier to inspect.
- Avoid decorative visuals.
- Use executive-friendly labels.
- Keep diagrams useful on mobile.
- Link every diagram to a template, framework, or brief.
