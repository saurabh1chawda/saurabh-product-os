This directory holds Mermaid source files that make Product Leadership Briefs easier to understand, facilitate, and reuse.

Available diagrams:

- [Decision Timeline](decision-timeline.mmd)
- [Stakeholder Map](stakeholder-map.mmd)
- [Trade-off Matrix](trade-off-matrix.mmd)
- [Decision Evolution](decision-evolution.mmd)

## Rendering Guidance

These files use Mermaid syntax and can render in:

- GitHub Markdown previews.
- Mermaid Live Editor.
- Documentation sites with Mermaid enabled.
- Internal wiki tools that support Mermaid.

Recommended export formats:

- SVG for documentation.
- PNG for presentations.
- Mermaid source for repository maintenance.

Diagram standards:

- Make the decision easier to inspect.
- Avoid decorative visuals.
- Use executive-friendly labels.
- Keep diagrams useful on mobile.
- Link every diagram to a template, framework, or brief.
