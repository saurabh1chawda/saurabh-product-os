# Roadmap

Status: v1 foundation  
Last updated: July 2026

## Version 1: Brief System Foundation

Goal: Establish Product Leadership Briefs as a coherent reusable decision library.

Scope:

- Flagship README
- Brief library navigation
- Decision index
- Product Decision Framework
- Leadership Brief template
- Architecture document
- Governance document
- Product specification
- Roadmap

Exit criteria:

- A visitor can understand what the repository is and how to navigate it.
- Briefs can be browsed by decision category.
- The Product Decision Framework is reusable.
- The Leadership Brief template can be copied into real product work.

## Version 2: Evidence Packs

Goal: Make each flagship brief easier to inspect and reuse.

Scope:

- Evidence packs for each brief
- Impact dashboard patterns
- Executive snapshot model
- Decision matrix examples
- Product review facilitation notes

Exit criteria:

- Each available brief has a supporting evidence pack.
- Visitors can compare briefs by decision type, trade-off, and outcome.
- Product Managers can use the format for their own product reviews.

## Version 3: Brief Creation Kit

Goal: Turn the repository into a public toolkit for writing executive product decision briefs.

Scope:

- Domain-specific variants for AI, platform, growth, payments, and enterprise SaaS
- Public examples that other PMs can adapt
- Workshop-ready decision review kit
- Portable Markdown and document-ready formats

Exit criteria:

- A team can run a product decision review using the repository without needing Product OS.
- The brief format works across multiple domains and decision types.
